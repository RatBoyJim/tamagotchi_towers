Tamagotchi Towers

Info about the Tamagotchi Towers project

Group Project | Javascript - React - Java Spring | February 2020




A full-stack browser game based on the wonderful world of Tamagotchi

  • Users can create a username and password to be able to create their own stable of Tamagotchis
  • Users can pick and name a Tamagotchi from one of six preset animal types that each have different and unique charachteristics
  • The user then has to keep their animal happy by feeding, exercising and cleaning their pet
  • Difficulty level can be increased for a greater challenge
  • User and Tamagotchi information is saved on a Java/Spring back end
